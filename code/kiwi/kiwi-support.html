The Interface

Locations

    The location manager is where you can view a list of all your trade zones, study sites, and current stores. You can see detailed information for each location as well as see where they are on the map. You can also add new study sites.

    Selecting a City - View Steps

    Locations are grouped by city. To select a city whose locations you want to view click on the select city drop down.

    Adding a New Location - View Steps

    1. Click on Add Location.
    2. Choose whether you want to add a single site or multiple sites.
      1. Adding a Single Site - View Steps
        1. Click Single Site.
        2. Fill in basic information.
        3. Find your location on the map by address, latitude/longitude or map click.
        4. Click Locate.
        5. Define service area by either radius or hand draw.
        6. Optionally add details and/or files.
        7. Click Save.
      2. Adding Multiple Sites - View Steps
        1. Select Multiple Sites.
        2. Click Browse.
        3. Locate file and click Open.
        4. Click Save.
      3. Updating Attributes for Sites - View Steps
        1. Select Update Attributes for Sites.
        2. Click Browse.
        3. Locate file and click Open.
        4. Click Save.

    Viewing Trade Zones - View Steps

    Click on the Trade Zones button to view trade zones in the manager list below.

    Viewing Study Sites - View Steps

    Click on the Study Sites button to view study sites in the manager list below.

    Viewing Stores - View Steps

    Click on the Stores button to view stores in the manager list below.

    Viewing Location Details - View Steps

    Click on the Info button of desired location to view details.

Data Layers

    The Data Layers panel allows you to view demographics, points of interest, and locations. Users can toggle data layers on and off individually or by group.

    Viewing Data - View Steps

    1. Click on the "plus" to open a data layer. To activate the layer make sure the check box is marked.
    2. Check the boxes of the data you want to view.

  • View Steps

Market Explorer - View

    Market Explorer allows you to search for specific areas of importance on the map, either by population, points of interest, or a combination of the two. Use Market Explorer to locate areas that meet specific brand criteria.

    Exploring Demographic Segments - View Steps

    1. Click Demographic Segment.
    2. Search by Grid or District.
    3. Adjust population levels.
    4. Click Search. A list of search results with scores will appear in the next panel. From here, you can click on a location name to view it on the map and add it to the location manager by clicking + Add Location. Users can refine search criteria by clicking on Refine.

    Exploring Points of Interest - View Steps

    1. Click Point of Interest.
    2. Search by Grid or District.
    3. Select a group.
    4. Select a category.
    5. Check individual brands to view or check the Select All Brands box.
    6. Set the minimum count to the desired level.
    7. Click Search.

    Exploring All - View Steps

    1. Click Explore All
    2. Search by Grid or District.
    3. Adjust population levels and click Next.
    4. Search by Grid or District.
    5. Select a group.
    6. Select a category.
    7. Check individual brands to view or check the Select All Brands box.
    8. Set the minimum count to the desired level.
    9. Click Search. A list of search results with scores will appear in the next panel. From here, you can click on a location name to view it on the map and add it to the location manager by clicking + Add Location. Users can refine search criteria by clicking on Refine.

Performance Viewer - View

    Quickly view the performance of all the stores in a particular city district or trade zone. View store revenue figures and performance breakdowns by store types.

    Viewing Performance by District - View

    1. Click the Select Territory tab arrow.
    2. Click District.
    3. Click the Select Year tab arrow.
    4. Click a year.
    5. Click an area on the map.

    Viewing Performance by Trade Zone - View

    1. Click the Select Territory tab arrow.
    2. Click Trade Zone.
    3. Click the Select Year tab arrow.
    4. Click a year.
    5. Click an outlined trade zone on the map.

Settings - View

    Upload and update stores, set POI variables, define sales territories, and manage users of the system.

    Uploading Store Information - View Steps

    1. Click Add Multiple Stores or Update Attributes for Stores.
    2. Click Browse.
    3. Select file to upload.
    4. Click Save.

    Setting POI Variables - View Steps

    1. Click Competitor or Synergy Group.
    2. Click the Select Group tab arrow.
    3. Click a group.
    4. Click the Select Category tab arrow.
    5. Click a category.
    6. Click checkboxes of desired brands.
    7. Click Add to List.
    8. Click Save.

    Defining Sales and Management Territories - View Steps

    1. Enter a territory I.D.
    2. Enter a territory name.
    3. Enter any territory descriptions.
    4. Select a method to define the territory.

    Defining Market Variables - View Steps

    1. Enter the desired values for each variable.

    Managing Users

      1. Adding Users - View Steps
        1. Click Add.
        2. Enter login information.
        3. Enter user information.
        4. Click Add User.
      1. Editing Users - View Steps
        1. Click on a user in the current user list.
        2. Click Edit.
        3. Modify login and/or user information.
        4. Click Modify.
      1. Deleting Users - View Steps
        1. Click on a user in the current user list.
        2. Click Delete.

Map Toolbar

    Use the toolbar to quickly navigate the map and add sites. You can also take snapshots and print areas of the map.

    Using the Hand Tool - View Steps

    Click on the hand icon in the toolbar. Pan the map by clicking on the map and moving the mouse.

    Zooming - View Steps

    Zoom in or out of the map with the scroll function of your mouse. Zoom in to a specific area with the zoom in tool by clicking on the map and dragging to create a desired zoom in area. Use the zoom out tool the same way.

    Map View Toggling

    Use the slider to toggle between street and satellite views.

    Creating Screen Captures - View Steps

    1. Click the camera button.
    2. Click the green rectangle tool.
    3. Drag out an area you want to capture.
    4. Save. Alternatively you can simply click the save button to take a screen capture of the entire map.

    Screen Capture Settings

    You can choose to save captures as either .jpeg or .png images as well as set the image quality for jpegs. You can also export images as a zip file that includes an additional coordinates file.

    Printing Maps - View Steps

    1. Click the print button.
    2. Add a title and description.
    3. Click Print.

    Using the Map I.D. Tool - View Steps

      1. View details about specific points on the map.
        1. Click the arrow icon.
        2. Click on an area or point on the map.
        3. View information in the identify panel.

    Adding Locations - View Steps

      1. Add new sites directly from the toolbar.
        1. Click the pin icon.
        2. Choose whether you want to add a single site or multiple sites.
      1. Adding a Single Site - View Steps
        1. Click Single Site.
        2. Fill in basic information.
        3. Find your location on the map by address, latitude/longitude or map click.
        4. Click Locate.
        5. Define service area by either radius or hand draw.
        6. Optionally add details and/or files.
        7. Click Save.
      1. Adding Multiple Sites - View Steps
        1. Select Multiple Sites.
        2. Click Browse.
        3. Locate file and click Open.
        4. Click Save.
      1. Updating Attributes for Sites - View Steps
        1. Select Update Attributes for Sites.
        2. Click Browse.
        3. Locate file and click Open.
        4. Click Save.

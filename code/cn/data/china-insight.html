中国透视

我们已经创建了 50 多个城市的标准化数据集，
这将有助于您的销售、营销和扩张规划。

你可以用这些数据完成什么？

寻找目标客户、优先考虑地域以及对市场数据的存储性能比较。

无论你是一个新的市场进入者，或有在中国现有的网络，这些数据将帮助验证您的销售、营销、扩张的决定，并更好地管理业务的风险。

用它来估计总市场潜力及净市场潜力的地区和重点商圈（从目标消费者、竞争对手、协同品牌派生出来）。通过比较市场数据，用它来更好地了解和调整您现有的店铺网络通过比较店铺的业绩和运营成本（POS 和固定/可变成本）。

你如何使用这些数据产品？

分析这些数据可以通过 GIS 工具，如 ESRI 的 ArcGIS、 MapBox 的 Tile Mill、 BI 软件或 MS Excel。

销售，市场营销或业务分析师可以插入标准化的工具，这个数据客用来支持分布规划，销售地区的开拓，和其他活动的营销活动策划。

范例－比方说，你是一个服装零售品牌，你想通知您的客户年终销售情况。这些数据加载到 Excel 电子表格，你可以很快发现大量目标消费者的领域。使用此信息来投资在派发宣传单张，基于位置的短信，和其他的营销活动的成本。
